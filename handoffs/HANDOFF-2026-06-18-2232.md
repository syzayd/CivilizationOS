### Frontend render (blank page debugging — ~45 min)
| Approach Tried | Why It Failed | Fix |
|---|---|---|
| Writing errors to `#root` in an index.html catcher | Corrupted React's container → cascading `removeChildFromContainer` masking the real error | Use a separate sink / ErrorBoundary instead |
| MCP `browser_execute_script` to read page `window.*` globals | Runs in an **isolated world** — shares DOM but not page JS globals (store invisible) | Verified via DOM queries + temporary default selection instead |
| MCP `browser_click(x,y)` on the Pixi canvas | Synthetic click didn't reach Pixi's federated event system (reported clicking canvas center) | Couldn't verify click→select via tool; verified inspector by temporarily defaulting `selectedId` |

**Real root cause of the blank page**: zustand footgun in `EventFeed.tsx` —
`useWorld((s) => s.world?.events ?? [])` created a **new `[]` every render**, so
zustand's identity check looped forever ("Maximum update depth exceeded"). Fix: move
the `?? []` **outside** the selector. Lesson: never return a fresh object/array from a
zustand selector.

### React StrictMode + Pixi
- StrictMode double-invokes effects in dev, racing the imperative Pixi canvas
  lifecycle. Dropped StrictMode in `web/src/main.tsx` (prod never double-invokes).

## Key Decisions
- **3-tier brain router, local-first**: routine agent life on Ollama (Qwen2.5 3B),
  Claude reserved for Phase-2 council debates. Keeps the whole sim at ~$0. Trade-off:
  3B dialogue is simpler than Claude's — acceptable for ambient citizen chatter.
- **Event-driven, non-blocking LLM**: conversation/reflection generation runs as
  background asyncio tasks; movement stays smooth, costs stay bounded.
- **numpy vector store over Chroma**: avoids Windows native-build pain, and gives full
  control over TCMF scoring in Phase 2.
- **In-memory sim (no DB yet)**: SQLite/persistence deferred; in-memory is fine for now.
- **GitHub: public repo `syzayd/CivilizationOS`**, one clean commit per phase
  (squashed the noisy auto-commits).
